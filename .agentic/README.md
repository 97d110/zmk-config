# Agent Context

This directory keeps the minimum operational context needed to work efficiently in this repo.

- `context/repo-map.md`: the maintained repo map and ownership boundaries.
- `commands.md`: canonical local commands for verification, builds, flashing, and debug logging.
- `checklists/change.md`: default handoff checklist.
- `tasks/TEMPLATE.md`: short task template for future work items.

Keep these docs short and aligned with the actual repo structure.
